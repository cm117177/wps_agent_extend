WPS AI 助手  使用说明
============================

【第一次安装 - 共 5 步】

  1. 解压收到的压缩包
     放到一个固定位置，例如 D:\WPS_AI_助手\
     （之后不要移动文件夹）

  2. 双击  install.bat
     按屏幕提示等待完成

  3. 完全退出 WPS Office
     任务栏右下角找到 WPS 图标 → 右键 → 退出

  4. 重新打开 WPS
     顶部菜单会多出「AI 助手」选项卡
     点击选项卡 → 点击「AI 助手」按钮

  5. 点击侧边栏右上角的齿轮图标 ⚙
     填入 API Key → 点保存

  安装完成！以后开机 AI 服务自动启动，
  无需任何额外操作。


【获取 API Key】

  推荐使用 DeepSeek（价格低，中文效果好）：

    1. 浏览器打开  platform.deepseek.com
    2. 注册账号
    3. 左侧菜单点「API Keys」→「创建 API Key」
    4. 复制生成的 Key（sk-开头的一串字符）
    5. 粘贴到 WPS AI 助手的设置里

  API 地址填：  https://api.deepseek.com/v1
  模型填：      deepseek-chat


【日常使用】

  文字 / 表格 / 演示文档：
    - 选中一段文字，然后在 AI 助手里提问
    - AI 会给出操作代码，点「执行」直接修改文档
    - 操作有误可点「↩ 撤销」还原

  PDF 文件：
    - 打开 PDF 后点 AI 助手，弹出浏览器窗口
    - 自动读取 PDF 内容，直接提问即可


【遇到问题】

  AI 助手选项卡消失了
    → 重新双击 install.bat，再重启 WPS

  侧边栏显示「连接失败」
    → 双击 install.bat 重新启动服务

  想要卸载
    → 双击 uninstall.bat，再重启 WPS
